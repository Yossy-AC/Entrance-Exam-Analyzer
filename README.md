# Entrance Exam Analyzer

塾の合格実績Excelファイルを自動集計・分析する Web ダッシュボード。

FastAPI + htmx + Chart.js 構成で、Excelファイルをドラッグ＆ドロップするだけでリアルタイムに分析レポートを確認できます。

## 機能

| ページ | 内容 |
|---|---|
| ダッシュボード | 合格者数・合格率サマリー、分類別/国公私別グラフ |
| 合格実績集計 | 受験方式別出願・合否、大学別合格者数ランキング |
| 共通テスト分析 | 文理別平均点、科目別グラフ、得点分布、合格者vs不合格者比較 |
| 経年比較 | 複数年度のExcelを読み込んだ合格者数推移グラフ |

## セットアップ

### 必要要件

- Python 3.12+
- [uv](https://docs.astral.sh/uv/)

### インストール

```bash
git clone https://github.com/Yossy-AC/Entrance-Exam-Analyzer.git
cd Entrance-Exam-Analyzer
uv sync
```

### 設定

`.env` ファイルをプロジェクトルートに作成し、Excelファイルのパスを設定します。

```
EXCEL_2026=C:\path\to\results_2026.xlsx
# EXCEL_2025=C:\path\to\results_2025.xlsx  # 経年比較用（任意）
```

> **注意**: パス・ファイル名は ASCII のみを使用してください。日本語を含むパスはファイルが正常に読み込まれない場合があります。

> `.env` を設定しなくても、ブラウザからExcelファイルをドラッグ＆ドロップして読み込めます。

## 起動

```bash
uv run uvicorn src.goukaku_analytics.main:app --host 127.0.0.1 --port 8000 --reload
```

ブラウザで http://localhost:8000 を開いてください。

## Excelファイル形式

以下のシート構成に対応しています。

| シート | 内容 |
|---|---|
| `2026合格実績(全体)` | 集計サマリー（読み取り専用） |
| `集計(全体)入力用` | **メインデータ**。1出願1行形式、ヘッダー5行目・データ6行目〜 |
| `学部学科等一覧` | 大学・学部マスター |

主要カラム（位置インデックスで読み込み）:

| 列 | 内容 | 値 |
|---|---|---|
| 15 | 合格 | 0=出願, 1=合格, 2=不合格 |
| 19 | 分類 | 1=国立, 2=公立, 3=大学校, 4=私立, 5=看護専門… |
| 21 | 大学名 | |
| 41 | 合計得点 | 共通テスト合計 |
| 44 | 文理区分 | |

> 「氏名」「ｼﾒｲ」列を削除したため、旧インデックスから -2 シフト済み。
